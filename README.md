# Empire 365

A business simulation game built with Python and Pygame where you build and manage your empire over 365 days. Make strategic decisions, manage resources, and grow your wealth from the ground up.

---

## 🎮 Gameplay Overview

You start with a set amount of capital and must make daily business decisions over the course of 365 in-game days. Every choice affects your bottom line — invest wisely, manage expenses, and outpace the competition to build your empire.

---

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| Python 3 | Core language |
| Pygame | Game engine / graphics |
| Pillow (PIL) | Image handling |

---

## 🚀 How to Run

### Prerequisites
Make sure you have Python 3 installed, then install the required libraries:

```bash
pip install pygame pillow
```

### Run the game
```bash
python empire_365_complete.py
```
